Variables

A variable is a container (storage area) used to hold data. Each variable should be given a unique name (identifier). Variables in Python can be created by simply assigning a value to a name.

Example:

x = 10  # Here, x is a variable storing the integer 10

Variable names can include letters, numbers, and underscores, but cannot start with a number.

Data Types

Data types are declarations for variables. This determines the type and size of data associated with variables which is essential to know since different data types occupy different sizes of memory.

There are 2 types of Data Types:

Primitive Data Types: To store simple values
Non-Primitive Data Types: To store complex values

Common Data Types:

  • int: Whole numbers, e.g., x = 5
  • float: Decimal numbers, e.g., y = 5.0
  • str: Strings (text), e.g., z = "Hello"
  • bool: Boolean values (True/False), e.g., w = True
Data Type Meaning Example
int Whole numbers x = 5
float Decimal numbers y = 5.0
str Textual data name = "Alice"
bool True or False is_active = True

Conditional Statements

Conditional statements allow you to execute certain pieces of code based on conditions.

Common Types:

  • if statement: Executes a block of code if a condition is true.
  • if-else statement: Executes one block if the condition is true, otherwise executes another block.
  • if-elif-else statement: Checks multiple conditions.

Example:

age = 20
if age >= 18:
    print("Adult")
else:
    print("Minor")

This checks if the age is 18 or above, and prints "Adult" or "Minor" accordingly.

Loops

Loops are used to execute a block of code repeatedly. The main types of loops are:

  • For Loop: Iterates over a sequence (like a list, tuple, or string).
  • While Loop: Repeats as long as a specified condition is true.
  • Nested Loop: A loop inside another loop.

Example of a For Loop:

for i in range(5):
    print(i)  # Prints numbers from 0 to 4

This loop prints numbers from 0 to 4.

Functions

A function is a block of reusable code that only runs when it is called. Functions can take inputs and return outputs. They help break the program into smaller, manageable parts.

Example:

def greet(name):
    return "Hello, " + name

print(greet("Alice"))  # Outputs: Hello, Alice

List

A list is a collection that is ordered and changeable. It allows duplicate members and is defined using square brackets.

Example:

fruits = ["apple", "banana", "cherry"]

You can access list items by their index:

print(fruits[0])  # Outputs: apple

Tuple

A tuple is a collection that is ordered and unchangeable. It allows duplicate members and is defined using parentheses.

Example:

colors = ("red", "green", "blue")

Tuples cannot be modified:

print(colors[1])  # Outputs: green

Dictionaries

A dictionary is a collection that is unordered, changeable, and indexed. No duplicate members are allowed. It is defined using curly braces.

Example:

person = {"name": "Alice", "age": 25}

You can access dictionary values by their key:

print(person["name"])  # Outputs: Alice
